## Project Overview
This project performs exploratory data analysis on the Titanic dataset to
understand passenger survival patterns.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Files in this Repository
- Titanic_EDA.ipynb : Jupyter notebook with EDA code and analysis
- titanic.csv : Dataset used for analysis
- EDA_Report.pdf : Final summarized EDA report

## Key Insights
- Females had a higher survival rate than males
- First-class passengers survived more than third-class
- Younger passengers had better survival chances
